# 基于JCanvascript的一款小游戏

------

### 1.使用了flexible来进行屏幕适配，尺寸单位用rem，通过sass的函数进行px到rem的转换。
### 2.进场动画以及转场动画切换等基于CSS3动画制作。
### 3.DOM操作方面，自己对一些基础操作进行封装了一个类jquery库，同样用$符号进行调用，包含选择器、css操作、事件绑定、事件监听、显示隐藏效果、属性的获取更改等。
### 4.Canvas 动画方面基于 **JCanvascript库**，Canvas的动画库现在有很多，原本打算使用CreateJs，但是jCanvaScript语法简单，格式简洁，上手快，很适合快速开发，在对Canvas方法的封装的同时增加了事件、拖动机制等。初学者可以选择JCanvasscript进行辅助了解Canvas。但是JCanvasscript已经在几年前就停止了维护，所以以后应该还是会使用CreateJs来进行Canvas动画的开发制作。
